# Screenshots

## How to Add Dashboard Screenshot

1. **Open your app** at http://localhost:5173 (or https://www.mediamanagerdx.com)

2. **Log in** to your account

3. **Take a screenshot** of the dashboard page:
   - **macOS**: Press `Cmd + Shift + 4` then press `Space`, click on the window
   - **Windows**: Press `Windows + Shift + S` and select area

4. **Save the screenshot** as `dashboard.png` in this directory

5. **Recommended**: Use a browser extension or tool to capture a full-page screenshot showing:
   - The navigation bar with logo
   - Dashboard cards
   - Server status
   - Clean, professional view

## Alternative: Use a Screenshot Tool

For best results, use a tool like:
- **macOS**: CleanShot X, Skitch
- **Windows**: ShareX, Greenshot
- **Cross-platform**: Firefox/Chrome Screenshot tool (Shift+F2, type "screenshot --fullpage")

## Tips for a Great Screenshot

- Use a clean, well-lit interface
- Show the dashboard with real data if possible
- Hide sensitive information (server names, IP addresses)
- Capture at 1920x1080 or higher resolution
- Use the desktop mode for a professional look
